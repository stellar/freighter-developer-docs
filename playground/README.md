# Playground

Test Freighter's extension API methods directly in your browser. Make sure you have the [Freighter extension](https://chromewebstore.google.com/detail/freighter/bcacfldlkkdogcmkkibnjlakofdplcbk) installed before trying these.

## Methods

| Method | Description |
| --- | --- |
| [isConnected](playground/isConnected.md) | Check if Freighter is installed |
| [isAllowed](playground/isAllowed.md) | Check if your app is authorized |
| [setAllowed](playground/setAllowed.md) | Request authorization for your app |
| [requestAccess](playground/requestAccess.md) | Request the user's public key |
| [getAddress](playground/getAddress.md) | Get the user's wallet address |
| [getNetwork](playground/getNetwork.md) | Get the active network |
| [getNetworkDetails](playground/getNetworkDetails.md) | Get full network configuration |
| [signTransaction](playground/signTransaction.md) | Sign a transaction XDR |
| [signAuthEntry](playground/signAuthEntry.md) | Sign a Soroban authorization entry |
| [signMessage](playground/signMessage.md) | Sign an arbitrary message |
| [addToken](playground/addToken.md) | Add a Soroban token to the wallet |
| [WatchWalletChanges](playground/watchWalletChanges.md) | Monitor wallet state changes |
